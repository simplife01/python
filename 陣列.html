陣列(字串、串列、元組)

陣列(字串、串列、元組)

tags: Python

在python中，陣列大致分為3種形式：字串(str)、串列(list)、元組(tuple)，下方會做三者間的詳盡比較

何謂陣列

陣列其實就是由一串相連在一起的變數所構成

  • 陣列需要取名稱
  • 陣列中的每一格都有專屬編號(索引)
  • 陣列索引值由0開始

為何需要陣列

之前我們所學的資料儲存方式都是利用變數
每當我們需要一個變數我們都得宣告一次
但倘若哪天我們有上千上萬筆資料空間需求時呢?
除了要大量宣告變數外，變數命名也是非常令人頭疼的事
(例如：a1 = 0, a2 = -2, .... , a100 = 10) 是不是很容易命名到手軟呢~

這時候若能利用陣列來儲存，那就能避免大量變數命名的痛苦了

各式陣列的表示法

字串

s = "我是字串" # 字串的表示方式為：" "

串列

li = ['我', '是', '串', '列'] # 串列的表示方式為：[]

元組

tu = ('我', '是', '元', '組') # 元組的表示方式為：()

存取陣列中的值

陣列的存取方式都是透過[]

s = "make me great again"
li = ['make', 'me', 'great', 'again']
tu = ('make', 'me', 'great', 'again')

print(s[2], s[4]) # k
print(li[2], li[0]) # great make
print(tu[2], tu[1]) # great me

注意! 空格、特殊符號都算一種字元唷!

I. 字串陣列

A. 字串切片(String Slicing)

從編號2號的格子到編號4號的格子

s = "Angels Like You"
print(s[2:5]) # gel

從編號0號的格子到編號14號的格子，取樣間隔為2

s = "Angels Like You"
print(s[0:15:2]) # Agl ieYu

如果右邊不放的話就代表到最末一個編號

s = "F**kin’ Perfect"
print(s[2:]) # *kin’ Perfect

如果左邊不放的話就代表從編號 0 開始

s = "世界が終るまでは"
print(s[:4]) # 世界が終

D. 更改字串中某些子字串

更改特定位置裡的值

a = "電視裡我唱得用力，撕心裂肺誰又會在意"
print(a.replace(a[4], "罵")) # 電視裡我罵得用力，撕心裂肺誰又會在意

將字串裡的所有某子字串全改為另個子字串

a = "不要等對的人走進你的生命裡；要努力成為走進別人生命裡那個對的人"
print(a.replace("生命", "XX")) # 不要等對的人走進你的XX裡；要努力成為走進別人XX裡那個對的人

注意! python中，字串不得直接用 a[] = " " 的方式更改!

# 錯誤示範
a = "AIDS"
a[0] = 'B' # TypeError: 'str' object does not support item assignment
print(a)

E. 判斷某子字串是否有在某字串中

可用 in 或 not in

txt = "A beautiful woman wears a smile on her face, carries confidence on her shoulder, keeps courage in her heart, and holds destiny in her hand."

conclusion1 = "shoulder" in txt # 判斷 shoulder 有無在 txt 字串中
conclusion2 = "beautiful" not in txt # 判斷 beautiful 有無在 txt 字串中

print(conclusion1, conclusion2) # True False

F. 字串搭配for迴圈

s = "banana"
for x in range(len(s)): 
  print(s[x] + " ", end="") # b a n a n a

for x in "banana": # x變數會從字串陣列中的第一格數值開始到最後一格
  print(x + " ", end="") # b a n a n a

G. 字串與變數或數字串接

name = "Lance"
age = 18
print('the age of {} is {}'.format(name, age)) # the age of Lance is 18

也可以簡化為下方形式

name = "Lance"
age = 18
print(f'the age of {name} is {age}') # the age of Lance is 18
print(f'the age of{name}is{age}') # the age ofLanceis18

print("the age of {} is {}".format("Lance", 18)) # the age of Lance is 18
print(f'the age of {"Lance"} is {18}') # the age of Lance is 18

也可以將要放的地方做編號(從0開始)

print("大家好，我是{0}，我今年{1}歲，身高{2}公分".format("皮卡丘", 15, 50)) # 大家好，我是皮卡丘，我今年15歲，身高50公分
print("大家好，我是{1}，我今年{2}歲，身高{0}公分".format("皮卡丘", 15, 50)) # 大家好，我是15，我今年50歲，身高皮卡丘公分

H. 格式化數字

格式化數字【放在{}中】有很多種類，在此只舉一些較常用的

輸入值 格式 輸出 解釋
3.1415926 {:.2f} 3.14 .2表示保留小數點後2位(4捨5入)
f表示float(小數)
13 {:x>5d} xxx13 x表示用x字元填補
>表示填補左方(向右對齊)
d表示decimal(10進位數字)
5d表示格式寬度為5個字元

5 {:!^5d} !!5!! !表示用!填補
^表示置中對齊
A {0:a^5} aaAaa 0表示此{}的編號

II. 串列 (可變動陣列List)

A. 串列宣告

thislist1 = [] # []
thislist2 = list() # []
thislist3 = list(("apple", "banana", "cherry")) # ['apple', 'banana', 'cherry']
thislist4 = ["apple", "banana", "cherry"] # ['apple', 'banana', 'cherry']

B. 存取串列中的值

跟字串陣列不同的地方在於能夠直接指定串列中特定格的值

thislist = ["apple", "banana", "cherry"]
thislist[1] = "blackcurrant"
print(thislist) # ['apple', 'blackcurrant', 'cherry']

串列中兩值互換位置

方法一：

a, b = 10, 5
tmp = a
a = b
b = tmp
print(a, b) # 5 10

方法二：

a, b = 10, 5
a, b = b, a
print(a, b) # 5 10

C. 串列搭配for迴圈

thislist = ["apple", "banana", "cherry"]
for x in thislist:
  print(x) # apple banana cherry

補充

lt = [1, 2, 3, 4, 5]
lt = [element - 1 for element in lt]
print(lt) # [0, 1, 2, 3, 4]

F. 陣列長度

利用len(<list_name>)就能得到陣列的長度值囉!

thislist = ["apple", "banana", "cherry"]

G. 加物件進陣列

可以利用append(<object>)方式加物件進陣列最末一個位置

thislist = ["apple", "banana", "cherry"]
thislist.append("orange")
print(thislist) #['apple', 'banana', 'cherry', 'orange']

也可以利用insert(<position>, <object>)方式加物件進陣列中的特定位置

thislist = ["apple", "banana", "cherry"]
thislist.insert(1, "orange")
print(thislist) #['apple', 'orange', 'banana', 'cherry']

也可以單純利用+=方式加物件進陣列中

thislist = ["apple", "banana", "cherry"]
thislist += "bike"
print(thislist) #['apple', 'banana', 'cherry', 'b', 'i', 'k', 'e']

thislist = ["apple", "banana", "cherry"]
thislist += ["bike"]
print(thislist) #['apple', 'banana', 'cherry', 'bike']

thislist = ["apple", "banana", "cherry"]
thislist += [["bike"]]
print(thislist) #['apple', 'banana', 'cherry', ['bike']] (變二維陣列)

複製整個陣列

注意! Python中不得直接指定(reference)整個陣列到另一個，這會使得更動一個陣列另一個也會被動到

排序陣列中的值

利用sorted(<list_name>)或.sort()將陣列中的值自動由小到大排列

num_list = [65, 20, 35, 100, 90, 36]
thislist = ["cherry", "banana", "apple", "orange", "kiwi", "melon", "mango"]

print(sorted(num_list))
print(sorted(thislist))

若希望由大至小排列可以加上reverse = True

num_list = [65, 20, 35, 100, 90, 36]
thislist = ["cherry", "banana", "apple", "orange", "kiwi", "melon", "mango"]

print(sorted(num_list, reverse = True)) #[100, 90, 65, 36, 35, 20]
print(sorted(thislist, reverse = True)) #['orange', 'melon', 'mango', 'kiwi', 'cherry', 'banana', 'apple']

III. 不可變動陣列

何謂不可變動陣列?

其實Tuple也是一種陣列，功能與使用方式與List有9成以上類似，不過特殊之處是不能輕易變動Tuple中的值

為何需要Tuple?

有些程式是需要存取陣列中固定某格當中的值，這時若用List來儲存可能會不小心更動到該格的數值進而影響結果，這時使用Tuple就能夠避免這類問題發生，因為Tuple是不可被變動的

創造Tuple

基本上Tupele的創造跟List的差別就在小括號與中括號的不同

thistuple = ("apple", "banana", "cherry") # Tuple是小括號
print(thistuple) # ('apple', 'banana', 'cherry')

不過如果Tuple中的值只有一個的情況下

thistuple = ("apple",) # 記得最後面要有個逗號，否則會跟字串搞混
print(type(thistuple)) # <class 'tuple'>

thistuple = ("apple") # 這就是字串
print(type(thistuple)) # <class 'str'>

無法直接改動Tuple中的值

thistuple = ("apple", "banana", "cherry")
thistuple[1] = "orange" # TypeError: 'tuple' object does not support item assignment
print(thistuple)

thistuple = ("apple", "banana", "cherry")
del thistuple[1] # TypeError: 'tuple' object doesn't support item deletion
print(thistuple)

間接改動Tuple中的值

x = ("apple", "banana", "cherry")
y = list(x) # 先轉為list
y[1] = "kiwi"
x = tuple(y) # 再轉回tuple
print(x) # ('apple', 'kiwi', 'cherry')

x = ("apple", "banana", "cherry")
y = list(x) # 先轉為list
del y[1]
x = tuple(y) # 再轉回tuple
print(x) # ('apple', 'cherry')

其餘功能與List幾乎相同

但list能用的有的Tuple未必也能使用，以下是列出Tuple能用的

用迴圈存取Tuple中所有的值
thistuple = ("apple", "banana", "cherry")
for x in thistuple:
  print(x)
確認Tuple中是否存在想要的值
thistuple = ("apple", "banana", "cherry")
if "apple" in thistuple:
  print("Yes, 'apple' is in the fruits tuple")
確認Tuple存在多少筆值
thistuple = ("apple", "banana", "cherry")
print(len(thistuple))
合併多個Tuple
tuple1 = ("a", "b" , "c")
tuple2 = (1, 2, 3)

tuple3 = tuple1 + tuple2
print(tuple3) # ('a', 'b', 'c', 1, 2, 3)
