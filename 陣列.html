陣列(List)

陣列

tags: Python

還記得我們在字串單元當中有提到Python中的字串是由陣列組成的吧!
這邊就再幫大家稍微回憶一下

何謂陣列

陣列就是由一串相連在一起的變數所構成

  • 陣列需要取名稱
  • 陣列中的每一格都有專屬編號(索引)
  • 陣列索引值由0開始

為何需要陣列

之前我們所學的資料儲存方式都是利用變數
每當我們需要一個變數我們都得宣告一次
但若哪天我們需要成千上百筆資料空間需求時呢?
除了要大量宣告變數外，變數命名也是非常令人頭疼的事
(例如：a1=0, a2=-2, .... ,a123=10) 是不是很容易命名到手軟呢XD

這時候如果有陣列，那就能大大減少大量變數命名的痛苦了

陣列宣告

在Python中，陣列通常都是由一對中括號來表示

thislist = [] #[]

thislist = list() #[]

thislist = list(("apple", "banana", "cherry")) #注意!是兩層小括號
print(thislist) #['apple', 'banana', 'cherry']

thislist = ["apple", "banana", "cherry"] #['apple', 'banana', 'cherry']

存取陣列中的值

基本上跟字串陣列的用法幾乎一致

n[<start>:<stop>:<step>] #第一格放起始索引值，第二格放終止索引值，第三格放每次往後幾個索引值

thislist = ["apple", "banana", "cherry"]
print(thislist) #印出整個陣列
print(thislist[1]) #印出陣列中索引值1那格的值
print(thislist[-1]) #印出陣列中索引值倒數過來1號那格的值

thislist = ["apple", "banana", "cherry", "orange", "kiwi", "melon", "mango"]
print(thislist[2:5]) #印出索引值2~4號格子的值

thislist = ["apple", "banana", "cherry", "orange", "kiwi", "melon", "mango"]
print(thislist[:3]) #['apple', 'banana', 'cherry']
print(thislist[4:]) #['kiwi', 'melon', 'mango']
print(thislist[-4:-1]) #['orange', 'kiwi', 'melon']
print(thislist[1:6:2]) #['banana', 'orange', 'melon']

跟字串陣列稍微不同的地方在於能夠直接指定陣列中特定格的值

thislist = ["apple", "banana", "cherry"]
thislist[1] = "blackcurrant"
print(thislist) #['apple', 'blackcurrant', 'cherry']

陣列長度

利用len(<list_name>)就能得到陣列的長度值囉!

thislist = ["apple", "banana", "cherry"]
print(len(thislist)) #3

加物件進陣列

可以利用append(<object>)方式加物件進陣列最末一個位置

thislist = ["apple", "banana", "cherry"]
thislist.append("orange")
print(thislist) #['apple', 'banana', 'cherry', 'orange']

也可以利用insert(<position>, <object>)方式加物件進陣列中的特定位置

thislist = ["apple", "banana", "cherry"]
thislist.insert(1, "orange")
print(thislist) #['apple', 'orange', 'banana', 'cherry']

將陣列中物件移除

使用remove(<object>)可移除陣列特定物件

thislist = ["apple", "banana", "cherry"]
thislist.remove("banana")
print(thislist) #['apple', 'cherry']

使用pop(<object>)會移除陣列中最後一個物件

thislist = ["apple", "banana", "cherry"]
thislist.pop() 
print(thislist) #['apple', 'banana']

使用del <list_name>可移除陣列中特定索引值的物件 或 移除整個陣列

thislist = ["apple", "banana", "cherry"]
del thislist[0]
print(thislist) #['banana', 'cherry']

thislist = ["apple", "banana", "cherry"]
del thislist
print(thislist) #NameError: name 'thislist' is not defined

清空陣列中物件

使用.clear()可清除整個陣列中的所有物件
clear() 同 del thislist[:]

thislist = ["apple", "banana", "cherry"]
thislist.clear()
print(thislist) #[]

複製整個陣列

注意! Python中不得直接指定(reference)整個陣列到另一個，這會使得更動一個陣列另一個也會被動到

thislist = ["apple", "banana", "cherry"]
li_tmp = thislist #這種指定方式非常危險
print(li_tmp) #['apple', 'banana', 'cherry']
thislist[0] = "WOOOW" 
print(li_tmp) #['WOOOW', 'banana', 'cherry']

要複製陣列可以使用.copy()

thislist = ["apple", "banana", "cherry"]
mylist = thislist.copy()
print(mylist) #['apple', 'banana', 'cherry']

也可以使用list(<list_name>)

thislist = ["apple", "banana", "cherry"]
mylist = list(thislist)
print(mylist) #['apple', 'banana', 'cherry']

陣列合併

使用 + 號合併兩個陣列
Python陣列中允許同時存在多種形態的值

list1 = ["a", "b" , "c"]
list2 = [1, 2, 3]

list3 = list1 + list2
print(list3) #['a', 'b', 'c', 1, 2, 3]

print(type(list3[3])) #<class 'int'>
print(type(list3[0])) #<class 'str'>

排序陣列中的值

利用sorted(<list_name>)或.sort()將陣列中的值自動由小到大排列

num_list = [65, 20, 35, 100, 90, 36]
thislist = ["cherry", "banana", "apple", "orange", "kiwi", "melon", "mango"]

print(sorted(num_list))
print(sorted(thislist))

若希望由大至小排列可以加上reverse = True

num_list = [65, 20, 35, 100, 90, 36]
thislist = ["cherry", "banana", "apple", "orange", "kiwi", "melon", "mango"]

print(sorted(num_list, reverse = True)) #[100, 90, 65, 36, 35, 20]
print(sorted(thislist, reverse = True)) #['orange', 'melon', 'mango', 'kiwi', 'cherry', 'banana', 'apple']

陣列中最大值、最小值及總合

利用max(<list_name>) min(<list_name>) sum(<list_name>) 找出陣列中最大值、最小值及總合

num_list = [65, 20, 35, 100, 90, 36]
print(max(num_list)) #100
print(min(num_list)) #20
print(sum(num_list)) #346
