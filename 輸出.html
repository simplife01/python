輸出

tags: Python

俗話說，眼見為憑，所以在進入程式的新世界裡，我們一開始就是要先來見識程式碼如何操縱電腦的

I. 註解

不過在讓各位見識到輸出結果之前，要先讓你們知道程式碼中，註解的意思

註解：用於對程式碼進行說明，被視為註解的文字不會被電腦所執行(不影響執行)

A. 單行註解

# 後方的即為註解

下方是可執行的程式碼

print('Make Me Great Again')

下方是被註解起來的程式碼(不會被執行)

# print('Make Me Great Again')

所以你也可以這樣寫(完全不影響程式執行)

# ლ(´ڡ`ლ)ლ(´ڡ`ლ)ლ(´ڡ`ლ)
# 太神奇了傑克
# こんにちは

B. 多行註解

一對 ''' ''' 或 """ """ 便是用於多行註解

"""
你像窩在被子裡的舒服
卻又像風 捉摸不住
像手腕上散發的香水味
像愛不釋手的 紅色高跟鞋
"""

注意!當需要多行註解時，通常先將要註解的程式碼反白後，按住 Ctrl + / 即可完成，若需取消註解，只需再次操作相同步驟即可

II. 印出一行字串

  • print 後要加上一對小括號
  • 小括號中的雙引號或單引號表示要印出的字串
print("當傷痛是一種選擇的時候，其實是一種幸福")

print('One day you will realize that what is more important than making lots of friends is cherishing the ones that are real.')

注意!偶爾，會打出 ａｎｙｔｉｍｅ的樣貌，而非 anytime，這時表示你將輸入法切換到全形(一般為半形)，按shift + 空白建 即可切換

注意!有時候，你所打的字會吃掉後面的字，這時就是你按到了 Insert 鍵，再按一次 Insert 即可

注意!本網頁中的程式碼都一定要實際執行才能夠完全了解他們的意義。使用下方快捷鍵，會讓過程愉快許多 ^ ^
複製的快捷鍵：Ctrl + c
貼上的快捷鍵：Ctrl + v
剪下的快捷鍵：Ctrl + x
還原的快捷鍵：Ctrl + z

III. 印出多行字串

方法一：用多個 print

print("菸一支一支一支的點 酒一杯一杯一杯的乾")
print('請你要體諒我 我酒量不好賣給我衝康 (。´艸`。)')

方法二：用 ''' 或 """

print('''一段感情之所以會失敗，
往往是因為你愛上的不是他，
而是你心目中的那個他''')

IV. 一行內印出多個字串

print("天竺鼠車車", 'PUI PUI', '515151')

這種方式，會印出由空格隔開的形式

V. 印出四則運算結果

四則運算後的結果會是數字型態

print(3) # 3
print(3+3) # 6
print(3-3) # 0
print(3*3) # 9
print(3/2) # 1.5 直接會變為浮點數
print(3%3) # 0 取餘數部分
print(3**3) # 27 指數
print(3//3) # 1 取商數部分

VI. 字串與數字的搭配

觀察下方幾種不同寫法的結果

print('3'+'3') # 33
print('3', '3') # 3 3
print('3', 3) # 3 3
print('3+3') # 3+3
print('3'*3) # 333
print(3*3) # 9

下方為常見的應用錯誤

print('3' 3) # 兩者間沒有符號做連結
print('3' + 3) # 將加號用於字串與數字
print('3' * '3') # 將/,%,-,//,...用於兩字串間

VII. 跳脫字元

有些符號為跳脫字元，必須要在前面加上一個\才顯示得出來唷!

print("\\") # \
print("\'") # '
print("\"") # "

A. 常用功能型跳脫字元

\n：換列

\t：水平跳格

\b：退格

print('人生最困難的不是面對各種挫折打擊\n，而是面臨各種挫折困難，仍沒有失去對人世的熱情')
print('人生最困難的不是面對各種挫折打擊\t，而是面臨各種挫折困難，仍沒有失去對人世的熱情')
print('人生最困難的不是面對各種挫折打擊\b，而是面臨各種挫折困難，仍沒有失去對人世的熱情')

B. print 隱藏的換列字元

print 內建\n，因此執行完會換一行

# 嘗試執行下方程式碼
print('人之初')
print('')
print('性本善')
print()
print("性相近")

若不希望換行可以用下列作法

print('人生沒有不能回頭的路', end="")
print("有時你只是忘了還可以轉身")

print('人生沒有不能回頭的路', end="，")
print("有時你只是忘了還可以轉身")
